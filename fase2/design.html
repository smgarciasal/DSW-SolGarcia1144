PROPOSITO DEL SITIO

El proposito es dar a conocer sobre HTML y CSS basado en los conceptos sobre Flex Box y Listas enlaces de HTML. informace de sus multiples ventajas y como tener su propio Sitio Web para dar a conocer su negocio.

FLEX BOX

Es uno de los valores HTML5 para propiedas CSS display, nos permite maquinar nuestras paginas web es un sistema novedoso de elementos flexibles que llegaron con la idea de remplazar los sistemas de creación de diseño bastantes tradicionales que no encajan con los dispositivos actuales como lo son: sistemas de escritorio, dispositivos móviles y multiples resoluciones, con FlexBoox nos acostumbraremos a una mecanica mucho más potente, limpia y personalizable, en la que los elemnetos HTML se adaptan y colocan automaticamente y es más facil personalizar los diseños, flex está especialmente diseñado para crear mediante CSS, estructura de una sola dimensión.

    Qué podemos hacer con Flexbox

  • Diseño flexibles con menos código
  • Alinear los elementos de manera vertical y horizontal
  • Reordenar contenido sin tocar el HTML

LISTAS Y ENLACES DE HTML

Para crear listas HTML mediante las etiquetas hay que tener en cuenta que estas posee propiedades CSS especificadas para alterar el estilo o caracteristicas de las marcas o viñetas que se crean a la izquierda de los items de la lista, mediante la etiqueta HTML. No esta demás aclarar que auque muchos elementos de las paginas web son un tanto "indulgentes" en cuanto al diseño no es muy grave si no esta de todo bien pero las listas y enlaces son muy importes porque si no se hace bien puede causar problemas graves para los usuarios que van a utilizar nuestros sitios web. En cuanto a los enlaces tienen algunos requisitos basicos de acuerdo al estilo y a la espectativa de los usuarios ya que los enlaces con estilos mal diseñados pueden arruinar la experiencia que tendra el usuario en cuanto utilice los sitios web porque la gente deberá detenerse a pensar donde dar el clip en el peor de los casos el usuario no sabra cuales son los enlaces de una pagina web.
